egret资源新加载

发表于 2018-04-11

最新加加载资源配置文件方式

两种方式，首先：

await RES.loadConfig(“resource/default.res.json”, “resource/“);

观察函数RES.loadConfig(…)的返回值，可以发下他的返回值是一个Promise对象

另外一种方式：

RES.LoadConfig.then(()=>{
    console.log("config file load complete!");
}).catch((err)=>{
    console.log(err);
});

关于await 和 ansy的一些说明和彼此配合

用await放在函数之前，是因为函数返回了了一个Promise对象，就是修饰了Promise对象。所以await只是针对promise对象。这是第一点。

其次，我们使用的await的时候，是阻塞的。比如：

await RES.loadConfig("resource/default.res.json", "resource/");
this.nextLogic();

在执行这一句的时候，要执行完了才能执行this.nextLogic();。如果不想阻塞，就执行this.nextLogic();，可以采用如下asyn方式：

public asyn doSthAwait(){
    await RES.loadConfig("resource/default.res.json", "resource/");
}

...

this.doSthAwait();
this.nextLogic();

这样，程序会在调用了doSthAwait()之后，马上执行nextLogic()

获得加载资源组的进度

引擎提供了一个接口类型，这个接口可以帮助我们获得进度：

先看一下定义：（略过也可以，( • ̀ω•́ )✧

declare namespace RES {
    /**
     * Promise的回调函数集合
     */
    interface PromiseTaskReporter {
        /**
         * 进度回调
         */
        onProgress?: (current: number, total: number) => void;
        /**
         * 取消回调
         */
        onCancel?: () => void;
    }
}

我们查看新引擎的新建项目，如何用，三步走：

继承这个接口:

class LoadingUI extends egret.Sprite implements RES.PromiseTaskReporter {
}

利用这个接口:

const loadingView = new LoadingUI();
await RES.loadGroup("preload", 0, loadingView);

实现接口函数：

public onProgress(current: number, total: number): void {
    this.textField.text = `Loading...${current}/${total}`;
}

分析：在LoadingUI界面里面继承这个接口（很淫荡的做法），然后在调用加载资源组的时候，这个实现了这个接口的类型的对象，赋值给加载资源组函数的第三个形参。这样，引擎资源组的资源的时候，加载成功一个就去调用这个接口的onProgress()，那么我们可以在onProgress()实现我们的业务逻辑。

自定义加载类型特别说明，因为官方文档out啦

在查看egret文档的时候，我们总是看到一种写法是`@RES.mapConfig，用于自定义资源格式的加载，但是在新版本的引擎版本中，我们看到这部分的定义是写在了script\confg.ts`里面，并且采用比较常见的的写法：

typeSelector: (path) => {
    const ext = path.substr(path.lastIndexOf(".") + 1);
    const typeMap = {
        "jpg": "image",
        "png": "image",
        "webp": "image",
        "json": "json",
        "fnt": "font",
        "pvr": "pvr",
        "mp3": "sound",
        "zip": "zip",
        "sheet": "sheet",
        "exml": "text"
    }
    let type = typeMap[ext];
    if (type == "json") {
        if (path.indexOf("sheet") >= 0) {
            type = "sheet";
        } else if (path.indexOf("movieclip") >= 0) {
            type = "movieclip";
        };
    }
    return type;
}
